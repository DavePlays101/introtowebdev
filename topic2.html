Building My Home Media & Automation Cluster

I’ve combined multiple SBCs—Pi-4, Pi-3, Odroid C4—and an HP Mini-PC into a unified cluster handling:

Services

  • Pi-hole ad-blocking
  • Jellyfin media streaming (with hardware transcoding)
  • Mailcow mail server
  • OpenProject for Kanban boards

Networking

Apache reverse-proxy via Cloudflare DNS → internal XAMPP on port 80/443 → VMs. VLANS separate IoT traffic from core services.

Next Steps

Migrating more workloads off InnS04-4200 NAS into this cluster; planning to automate backups with rsync + Docker containers.
